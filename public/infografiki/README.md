## Zasady

- Format: PNG
- Proporcje: 3:2 (zalecane 1800 x 1200 px)
- Rozmiar pliku: poniżej 200 KB per plik (skompresuj w [squoosh.app](https://squoosh.app) jeśli trzeba)
- Białe tło (aby pasowało do stron)

Po wrzuceniu plików nic więcej nie musisz robić — Astro sam je pokaże przy kolejnym buildzie.
